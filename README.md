# Test React Native

Diseña en React Native una aplicación para la store de lego, para esta tienda requerimos un servicio de autenticación para nuestros clientes, usar el servicio proporcionado por Firebase para la autenticación, la tienda consumirá un API

[api](https://documenter.getpostman.com/view/25149686/2s8Z73zWXA)


## Requisitos:

- [x] Diseñar una pantalla de autenticación que consumirá el servicio de firebase
- [ ] Agregar 2 estilos de tema diferentes a esta pantalla de login (tema libre Ej: Dark y light)
- [x] Una vez me loguee la sesión debe persistir
- [x] Luego de iniciar sesión deberá mostrar la pantalla principal de la tienda
- [x] Esta pantalla mostrar una lista de productos que podría obtener del Mock Dummy proporcionado
- [x] En esta pantalla podría ver la foto de cada productos y agregarlos al carrito de acuerdo a su stock
- [x] Podrá seleccionar un producto y esto lo llevara a una nueva vista con el detalle de este productos (desde acá también lo podría agregar al carrito)
- [x] Finalmente el usuario podrá dar click en comprar esto enviara una petición al Api y actualizara el stock de los productos (nota el api es un mock dummy siempre tendrá la misma respuesta el objetivo es actualizar el stock con esa respuesta y no con los valores reales de la compra)
